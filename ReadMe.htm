Blender 2.5 to XNA 4 animated FBX model exporter

Download

Blender Wiki

These scripts provide versions of the Autodesk FBX exporter compatible with Microsoft XNA 4.

Abridged Installation

  • Copy the 'io_export_anim_mesh_xna' folder and contents in to the users Blender '.../scripts/addons' folder.
  • Open Blender and enable the script within the 'Addons' tab from File -> User Preferences.
  • Use 'Save Default' to have the script enabled whenever Blender starts.

For more detailed instructions see the Blender wiki pages.
