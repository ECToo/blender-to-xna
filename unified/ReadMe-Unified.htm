Unified FBX Exporter for Blender 2.5

Download

Blender Wiki

These scripts provide versions of the Autodesk FBX exporter compatible with Microsoft XNA 4 and others will ensure they are also compatible with Unity and the UDK.

The unified version is based on the revision 38691 dated 25 July 2011 about 15:30 UK time. The intention is to keep this up to date with all changes and then submit a patch when complete.

Dependencies:

  • modules\bpy
  • modules\bpy_extras

The release version 2.58a of Blender must have all the above modules updated to the same revision as the starting script in order for the script to run. Download the latest versions from the subversion tree:
'\blender\release\scripts\modules'

Abridged Installation

  • Copy the 'io_scene_fbx' folder and contents in to the users Blender '.../scripts/addons' folder.
    'C:\Program Files\Blender Foundation\Blender\2.58\scripts\addons\io_scene_fbx'
  • Contents of the existing sub-floders cache must be deleted for the new scripts to take effect.
  • The FBX script must be enabled in the User Preferences.

For more detailed instructions see the Blender wiki pages.
